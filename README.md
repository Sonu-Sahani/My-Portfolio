# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Sonu-Sahani/My-Portfolio.git
```

Navigate to the project directory

```bash
cd My-Portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
